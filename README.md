Data Science & Analysis Portfolio
Overview This portfolio contains a collection of projects demonstrating skills in data analysis, machine learning, and statistical modeling. The work highlighted below utilizes Python, R, and Apache Spark to solve problems ranging from economic analysis to biodiversity modeling.
Part I: Python & Machine Learning Projects These projects utilize Python libraries (pandas, scikit-learn) and Jupyter Notebooks for data modeling and analysis.
Childcare Cost Model
Description: An analytical project aimed at modeling and understanding trends in childcare costs. This work explores factors influencing pricing and affordability across different demographics and regions.
Tech Stack: Python, Jupyter Notebook
iNaturalist City Observation Model
Description: A predictive modeling project focused on urban biodiversity. It utilizes data from iNaturalist to analyze and predict the frequency of nature observations within city environments.
Tech Stack: Python, Jupyter Notebook
Car Price Model (SparkML)
Description: A script leveraging Apache Spark's Machine Learning library (SparkML) to predict car prices. This demonstrates the application of big data tools and scalable machine learning pipelines.
Tech Stack: Python, PySpark, SparkML
Housing Analysis
Description: An exploratory data analysis (EDA) project focused on housing market data. The project identifies key market drivers and potential trends in housing prices.
Tech Stack: Python, Jupyter Notebook
Movie Recommender
Description: A recommendation system designed to suggest movies to users, utilizing collaborative or content-based filtering techniques to personalize viewing options.
Tech Stack: Python, Jupyter Notebook
Text Sentiment Analysis
Description: A Natural Language Processing (NLP) project that analyzes text data to determine sentiment polarity (positive, negative, or neutral).
Tech Stack: Python, NLP Libraries
Weather API
Description: A Python script demonstrating the integration of external APIs. It fetches, parses, and utilizes meteorological data from a live weather service.
Tech Stack: Python, API Integration
Part II: R & Statistical Analysis Projects These projects utilize R for statistical computing and complex data visualization.
Analysis of Kia & Hyundai Theft
Description: A statistical analysis investigating the trends and variables surrounding the theft rates of specific vehicle manufacturers.
Tech Stack: R
Invasive Species and GDP
Description: A study exploring the correlation and potential economic impact of invasive species on Gross Domestic Product (GDP).
Tech Stack: R
Technical Skills Summary
Languages: Python, R
Big Data: Apache Spark (SparkML)
